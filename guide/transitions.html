# Transitions

WARNING

If you are not comfortable with transition in vue 3 please check this link first.

If you want to make some fancy open/close effects. vue3-treeview provide a transition on levels.

<template>
  <Tree :nodes="nodes" :config="config"/>
</template>

<script lang="ts">
import Tree from "@src/components/Tree.vue";

export default {
  components: {
    Tree,
  },
  data: function () {
    return {
      nodes: {
        id1: {
          text: "text1",
          children: ["id11"]
        },
        id11: {
            text: "text11"
        }
      },
      config: {
        roots: ["id1"]
      },
    };
  }
};
</script>
<style src="../style/transition.css"/>
.level-enter-active,
.level-leave-active {
  transition: opacity 0.5s ease;
}

.level-enter-from,
.level-leave-to {
  opacity: 0;
}
